# Fragment-M2-Notes

An interactive, lightweight digital notebook framework designed for mathematics education.

## Core Features

- **Dynamic Math Rendering**: Full support for LaTeX equations via MathJax with custom gradient styling for key terms.
- **Modular UI Components**: Pre-styled boxes for Theorems, Definitions, Examples, and Proofs, fully responsive across all devices.
- **Interactive GeoGebra Integration**: A custom loader ensures embedding of multiple, interactive graphs with real-time data tracking.
- **Smart Assessment Tools**: 
  - **MCQ Engine**: Box-style multiple-choice questions with instant feedback, LaTeX-supported explanations, and a "retry" mechanism to encourage learning.
  - **Collapsible Sections**: Collapsible containers for hints, proofs, or deep-dive content to keep notes organized.
